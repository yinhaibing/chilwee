# 超威电池开门红抽奖小游戏

一个完整的抽奖系统，包含前台抽奖界面和后台管理界面。

## 功能特点

### 前台抽奖界面
- 🎫 输入抽奖码启动抽奖
- 🎡 动态转盘抽奖动画
- 🏆 自动根据概率抽取奖品
- 📊 实时显示中奖结果
- 🖼️ 支持奖品图片展示
- 📱 响应式设计，支持移动端

### 后台管理界面
- 🔐 安全的管理密码验证
- 🎫 抽奖码管理（添加/删除）
- 🏆 奖品设置（名称/数量/概率/图片）
- 📊 概率实时计算和验证
- 📋 抽奖记录查看
- 📥 导出抽奖记录（CSV格式）
- 💾 本地数据持久化

## 使用说明

### 前台抽奖

1. 打开 `index.html`
2. 输入有效的抽奖码
3. 点击"开始抽奖"按钮
4. 等待转盘停止，查看中奖结果

### 后台管理

1. 打开 `admin.html`
2. 输入管理密码：`chaowei2026`
3. 验证成功后进入管理界面

#### 添加抽奖码
- 在"抽奖码管理"区域输入抽奖码
- 点击"添加"按钮
- 抽奖码会显示在列表中

#### 添加奖品
- 填写奖品名称、数量、中奖概率
- 上传奖品图片（可选）
- 点击"添加奖品"按钮
- 系统会自动验证总概率不超过100%

#### 查看记录
- 在"抽奖记录"区域查看所有中奖记录
- 点击"导出记录"可下载CSV文件
- 点击"清空记录"可删除所有记录

## 注意事项

1. 所有数据存储在浏览器本地，清除浏览器数据会丢失所有设置
2. 抽奖码使用后会被自动消耗
3. 奖品数量为0时不会被抽中
4. 总概率必须控制在100%以内
5. 建议定期导出抽奖记录备份
